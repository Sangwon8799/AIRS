# AIRS
**AIRS(AI Integrated Room System)** is a project team at Sogang University dedicated to developing an autonomous **intelligent air conditioning system**.
Our system maintains an **optimal environment** by analyzing user patterns and real-time indoor data (e.g. temperature, humidity) collected via integrated sensors.
