# AIRS
**AIRS(AI Integrated Room System)** is a project team in Sogang Univ. that is on duty for developing **intelligent air conditioner** which controls itself.
Intelligent air conditioner means it maintains **pleasant environment** for human by analyzing the patterns of user and the information(e.g. temperature, humidity) in the room where the sensors are built.
